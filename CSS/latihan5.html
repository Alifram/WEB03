Apa itu CSS

CSS atau Cascading Style Sheet adalah dokumen web yang memiliki fungsi mengatur tampilan dokumen web lin,yaitu HTML. Beberapa contoh pemanfaatan CSS pada HTML adalah untuk mengatur warna latar belakang. Gata huruf, jarak antar elemen, dan masih banyak lagi. CSS terdiri atas kumpulan kode-kode yang keberadaannya tidak menggantikan HTML menjadi lebih indah dan dinamis. Berikut ini adalah contoh dokumen CSS.

Bagaimana menggunakan CSS?

Ada 3 cara menggunakan CSS:

  1. Inline style: Menuliskan kode CSS langsung pada tag akan diubah atau diatur style-nya.
  2. Internal/Embedded Style: menuliskan kode CSS pada bagian header dokumen HTML
  3. Eksternal/Linked Style: Menuliskan kode CSS pada file terpisah dan dipanggil melalui tag link dari dokumen HTML
